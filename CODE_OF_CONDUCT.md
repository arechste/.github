# Code of Conduct

## Standard

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

## Summary

- Be respectful and constructive
- Focus on what's best for the project
- No harassment, discrimination, or personal attacks
- Accept constructive criticism gracefully

## Enforcement

Unacceptable behavior can be reported to the project maintainer. All complaints will be reviewed and investigated promptly and fairly.
